```# Medical Records Management System (Medical TV Show Theme)

A sleek, front-end only patient registration and management interface inspired by medical drama aesthetics. This project demonstrates core frontend skills, including form handling, data encryption simulation, and dynamic display of patient records.

---

## Features

- User-friendly patient registration form
- Responsive, themed UI inspired by medical TV shows
- Simulated data encryption/decryption (using Base64)
- Dynamic listing of registered patients
- Clear, modern design emphasizing accessibility and aesthetics

## Setup & Usage

### How to run locally

1. **Clone or download** this repository.
2. Open `index.html` in your preferred web browser.
3. Fill out the registration form with patient details.
4. Click **Register Patient** to add to the list.
5. View registered patients below the form.

### In CodePen

- Copy the **HTML**, **CSS**, and **JavaScript** code into respective panels.
- Save and run to see the interactive registration system with the themed UI.
